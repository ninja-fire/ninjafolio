Pozzle Planet

Description

Aramid Finance is a decentralized cross-chain protocol that enables the seamless transfer of assets across multiple blockchains. The platform utilizes bulletproof on-chain verification, a decentralized relayer node network, and a robust pipeline to ensure the security and convenience of its users. As a UX/UI designer, I was tasked with creating a user-friendly interface for the Aramid Finance platform that would make it easy for users to navigate and understand the various features and functionality of the platform.

Design Challenges

The main design challenge for this project was to create an interface that would be easy to use for users with varying levels of technical expertise. The platform had a lot of complex functionality, and we wanted to make sure that users could easily access and understand all of the features without feeling overwhelmed. Additionally, as the platform supported multiple blockchains, we needed to ensure that the user could easily interact with all of the different blockchain networks.

Design Process

To tackle these design challenges, we began by conducting user research to understand the needs and pain points of our target audience. We conducted user interviews and surveys to gather feedback and insights on how users currently interact with decentralized cross-chain protocols and what features they would like to see in a platform like Aramid Finance. Based on our research, we created a set of user personas and user scenarios to guide our design decisions. We also created a set of design principles that would guide the overall look and feel of the interface. Next, we created wireframes and mockups of the interface, and tested them with users to gather feedback and make iterative improvements. We also worked closely with the development team to ensure that the interface was technically feasible and compatible with the different blockchain networks.

Final Design

The final design of the Aramid Finance interface is clean, modern, and easy to navigate. The interface is divided into three main sections: "Bridge", "Transaction History", and "Claim". The "Bridge" section allows users to easily transfer assets between different blockchains. In the "Claim" section, users can view their bridged assets and claim them on the destination chain. The "Transaction History" section allows users to view a detailed record of all their past transactions. Additionally, the interface is fully responsive and works seamlessly on all devices. To simplify the current and future application developments, I have created a Design System in Figma, including Figma design tokens and components that can be used through all Figma design files.

Conclusion

The design of the Aramid Finance platform was a challenging but rewarding project. By conducting user research and working closely with the development team, we were able to create a user-friendly interface that makes it easy for users to navigate and understand the various features and functionality of the platform. The final design of the interface is modern, clean, and responsive, making it easy for users to access the platform from any device.
